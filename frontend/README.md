# AegisVault RAG Privacy Guard — Web Security Console

This is the modern, highly polished cybersecurity operations dashboard for **AegisVault RAG Privacy Guard**. It is built using **React**, **Vite**, **TypeScript**, and **Tailwind CSS**, with **Lucide React** for icons.

---

## Key Features

1. **Operations Dashboard**: Real-time health statistics of system modules (PostgreSQL, Redis, Neo4j Graph, ChromaDB Vectorstore, and LLM Client). Tracks compliance events and document volumes.
2. **Guarded Query Console**: Interactive interface to test and query the RAG system. Provides detailed security traces showing RBAC blocks, PII scrub volumes, canary leakage protection, and response latencies.
3. **Data Ingestion Portal**: Tabbed interface supporting text uploads and file parsing (`.txt`, `.md`, `.csv`, `.pdf`). Configures logical tenants and ACL parameters, with support for Celery task queuing.
4. **Security Audits & Traces**: Dynamic audit ledger displaying compliance events generated by user operations. Filterable by risk level (Low, Medium, High, Critical) and transaction type.
5. **Gateway Settings**: Secure management of endpoints and API keys (`X-API-Key`). Configurable directly in the UI and stored locally in browser storage for maximum confidentiality.

## Technical Setup

### Prerequisites

Ensure you have **Node.js** (v18+) and **npm** installed.

### Setup and Start Local Development

1. **Install dependencies**:
   ```bash
   cd frontend
   npm install
   ```

2. **Configure environment variables (Optional)**:
   Create a `.env` file from the example:
   ```bash
   cp .env.example .env
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   *The client will boot up (typically on `http://localhost:5173`).*

4. **Connect to the backend**:
   Open the application in your browser, navigate to the **System Settings** page, enter your RAG FastAPI base URL (e.g. `http://127.0.0.1:8000`), input your API security key, and click **Commit Changes**.

### Building for Production

Compile and validate the TypeScript bundle:
```bash
npm run build
```
This runs the TypeScript compiler `tsc` and bundles assets with Vite into `dist/`.
